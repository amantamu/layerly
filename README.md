# Layerly

A full-stack monorepo project built with Turborepo.

## Structure

- `apps/web` - Next.js frontend application
- `apps/server` - FastAPI backend application
- `packages/` - Shared packages for UI, core logic, types, utils, and config
- `docs/` - Documentation and specifications

## Getting Started

```bash
# Install dependencies
pnpm install

# Run all apps in development mode
pnpm dev

# Build all apps
pnpm build
```

## Tech Stack

- **Monorepo**: Turborepo
- **Package Manager**: pnpm
- **Frontend**: Next.js
- **Backend**: FastAPI
- **Language**: TypeScript (frontend), Python (backend)
